# Bun Project Setup - written by Salah

## IF YOU DONT HAVE NODEJS INSTALL IT FROM THEIR SITE THEN:

## Install dependencies

If you're using **npm**:

```bash
npm install --legacy-peer-deps
```

If you already have **Bun** installed:

```bash
bun install
```

## Run the development server

Using Bun:

```bash
bun run dev
```

Or with npm:

```bash
npm run dev
```
